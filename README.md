

# Responsive Landing Page

## 📌 Overview

This project is a Responsive Landing Page created during my Web Development Internship at Prodigy InfoTech. The main objective of this project is to build an interactive and visually appealing navigation menu with responsive design using HTML, CSS, and JavaScript.

The navigation bar changes style when the user scrolls through the page and includes hover effects for better user interaction.

---

## 🚀 Features

* Responsive design for desktop and mobile devices
* Interactive navigation menu
* Scroll-based navbar styling
* Hover effects on menu items
* Clean and modern user interface
* Smooth scrolling experience

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
